# Incognito RPG - 隐身笔记

一款披着极简笔记软件外衣，实则靠盲打键盘驱动剧情的文字冒险越狱游戏。

## 游戏简介

在课堂上，你假装在认真记笔记，但实际上正在策划一场惊心动魄的越狱行动！

- **伪装**：无论你敲什么键，屏幕上都会显示预先写好的笔记内容（高数、英语、物理、历史等）
- **真实**：右下角实时显示 RPG 进度，你需要通过不断敲击键盘来推进剧情
- **选择**：当敲击次数达到特定阈值时，会出现剧情分支选项，按数字键进行选择
- **结局**：做出明智的选择，成功逃脱或被抓住，两种结局等你来探索

## 技术栈

- **后端**：Python + FastAPI
- **前端**：原生 HTML/CSS/JavaScript
- **部署**：Uvicorn

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 启动服务器

```bash
python main.py
```

或者使用 uvicorn 直接启动：

```bash
uvicorn main:app --host 0.0.0.0 --port 2833 --reload
```

### 3. 访问游戏

打开浏览器访问：`http://localhost:2833`

## API 接口

### 获取笔记类型

```http
GET /api/note-types
```

返回所有可用的笔记类型。

### 开始游戏

```http
POST /api/start-game
Content-Type: application/json

{
    "note_type": "math"
}
```

开始新游戏会话，返回 session_id 和初始剧情节点。

### 处理键盘敲击

```http
POST /api/keystroke/{session_id}
```

处理一次键盘敲击，推进剧情进度，返回需要添加的笔记文本和新的进度状态。

### 做出剧情选择

```http
POST /api/make-choice/{session_id}?choice_key=1
```

在剧情分支点做出选择，推进到下一个剧情节点。

## 游戏玩法

1. **选择笔记类型**：游戏开始时，选择你想要"伪装"的笔记类型（高等数学、大学英语、大学物理、中国近现代史）

2. **盲打键盘**：进入游戏后，随意敲击键盘上的任何按键，屏幕上会匀速输出你选择的笔记内容

3. **观察进度**：留意右下角的 RPG 进度面板，它会显示当前的剧情状态和进度百分比

4. **做出选择**：当剧情到达分支点时，进度面板会显示选项，按对应数字键（1、2、3、4）做出选择

5. **追求完美结局**：明智的选择将帮助你成功逃脱，错误的选择可能导致被抓住

## 剧情概要

你是一名被困在枯燥课堂上的学生，正在策划一场越狱行动。

**剧情节点**：
- 节点 0-1：课堂中，等待机会
- 节点 2：老师要点名，选择应对方式
- 节点 3-5：继续等待或寻找机会
- 节点 6-12：离开教室，遭遇教导主任
- 节点 13-17：逃离教学楼，遭遇保安
- 节点 18-26：逃离学校，最终抉择
- 节点 27：成功逃脱！

**结局类型**：
- **成功**：🎉 成功逃离学校
- **失败**：😢 被抓住，逃课记过处分

## 开发说明

### 后端逻辑

- `NOTE_CONTENTS`：存储四种笔记类型的文本内容
- `STORY_NODES`：存储剧情节点数据，包括描述、进度、选项等
- `active_sessions`：存储活跃的游戏会话状态
- 每个键盘敲击会推进笔记内容输出和剧情进度

### 前端逻辑

- 键盘事件监听：捕获所有按键事件
- API 交互：与后端通信获取剧情和笔记数据
- UI 更新：动态更新编辑区内容和进度面板
- 动画效果：闪烁光标、平滑过渡等视觉效果

## 部署

服务器默认监听端口 **2833**。

如需修改端口，编辑 `main.py` 最后一行：

```python
if __name__ == "__main__":
    uvicorn.run(app, host="0.0.0.0", port=2833)
```
